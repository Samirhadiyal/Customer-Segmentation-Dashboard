## 📌 Overview

Customer Segmentation Dashboard is an end to end machine learning powered customer segmentation platform built using Streamlit and Scikit learn.

It transforms raw customer datasets into interactive visual insights and professional business reports.

Unlike basic clustering scripts, Customer Segmentation Dashboard provides a full analytical workflow including data validation, dynamic clustering control, intelligent naming and interactive dashboard visualization

Check out app:- https://customerseg-dashboard.streamlit.app/

- Unsupervised Learning  
- KMeans Clustering  
- Feature Scaling  
- Dynamic Cluster Selection  
- Data Quality Validation  
- Interactive Visualization

## 📊 Key Features

✔ Automatic Data Quality Report  
✔ Flexible Clustering (3, 5, 7 or Custom)  
✔ Interactive Pie and Bar Charts  
✔ Scatter Based Cluster Visualization  
✔ Intelligent Segment Naming  
✔ CSV Export

## 🗂 Dataset

Sample dataset used:

Mall Customers Dataset  

https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

Dataset must contain at least two numeric columns such as:

- Age  
- Annual Income  
- Spending Score
